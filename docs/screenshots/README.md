아래 파일명 그대로 저장해야 README 이미지가 표시된다.

| 파일명 | 화면 |
|---|---|
| `create.jpg` | 폴 생성 (홈) |
| `edit.jpg` | 내 가능 시간 (참가자 응답) |
| `group.jpg` | 그룹 현황 (히트맵) |

- 파일명을 바꾸면 루트 [README.md](../../README.md)의 `<img src>`도 함께 수정할 것.
